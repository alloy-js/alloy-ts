constructor

  • new AlloyField(name: string, types: Array<AlloySignature>, tuples: Array<AlloyTuple>, is_meta?: undefined | false | true, is_private?: undefined | false | true): AlloyField

  • Overrides AlloyElement.constructor

    • Defined in core/AlloyField.ts:29

    Create a new Alloy field.

    remarks

    An Alloy field consists of an ordered array of signatures that define the types of the "columns" of the relation defined by the field, as well as a list of tuples that define the contents (or "rows") of the relation. The arity of the field, or number of columns, must be at least two.

    throws

    Error When the arity is less than two or an atom's type does not match the type of the column in which it resides.

    Parameters

    • name: string

      The name of this field

    • types: Array<AlloySignature>

      The types that define the columns of this relation

    • tuples: Array<AlloyTuple>

      The contents of this relation

    • Optional is_meta: undefined | false | true

      Is this a meta field?

    • Optional is_private: undefined | false | true

      Is this a private field?

    Returns AlloyField
