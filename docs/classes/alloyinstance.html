  • new AlloyInstance(text: string): AlloyInstance
    • Defined in core/AlloyInstance.ts:21

    Assemble an Alloy instance from an XML document.

    remarks

    Extracts and parses all info from an instance that has been exported from Alloy in XML format. The plain text read from the XML file is passed to the constructor; all XML parsing is performed automatically.
