constructor

  • new AlloyInstance(document: Document): AlloyInstance
    • Defined in AlloyInstance.ts:19

    Assemble an Alloy instance from an XML document.

    remarks

    Extracts and parses all info from an instance that has been exported from Alloy in XML format. Typically, XML files are read in Javascript and the resulting document passed to this constructor as follows:

    let text = '...'; // The text read from the XML file
let parser = new DOMParser();
let doc = parser.parseFromString(text, 'application/xml');
let instance = new AlloyInstance(doc);

    Parameters

    • document: Document

      The XML document

    Returns AlloyInstance
