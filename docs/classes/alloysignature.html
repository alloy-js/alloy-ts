Class AlloySignature

A signature in an Alloy instance.

remarks

In Alloy, a signature introduces a set of atoms.

eachBefore

  • eachBefore(callback: function): AlloySignature
    • Defined in AlloySignature.ts:181

    Invokes the specified function for this signature and each descendant signature in pre-order traversal, such that a given signature is only visited after all of its ancestors have already been visited. The specified function is passed the current signature.

    remarks

    As an example, to print the nested signature hierarchy:

    let univ = ...; // retreive the univ signature
univ.eachBefore(sig => {
  console.log(' '.repeat(sig.depth()) + sig.name());
}

    Parameters

    • callback: function

      The function to call for each signature

        • (signature: AlloySignature): any

        • Parameters

          • signature: AlloySignature

          Returns any

    Returns AlloySignature

    This signature is returned, allowing for method chaining.
